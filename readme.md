## TL;DR
  Contacts list
## Setup

```bash
  # If you don't have expo installed
  npm install --global expo-cli

  # Install dependencies
  npm install
```

## Execute

```bash
  # Run in Andriod
  npm run android

  # Run in ios
  npm run ios

  # Run in web
  npm run web
```
